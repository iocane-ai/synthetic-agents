# Connect to Iocane ATI

Follow these steps to connect your local environment to Iocane ATI and visualize the agent behaviors.

### 1. Install Iocane CLI

```bash
curl -fsSL https://iocane.ai/install.sh | sh
```

### 2. Login

Make sure you are logged in to your Iocane account.

```bash
iocane login
```

### 3. Connect

Establish the connection to start sending traces.

```bash
iocane connect
```
